# Om Ansvarlig KI i skolen

**Veiviser for ansvarlig bruk av kunstig intelligens i skolekontekst**

## Hva er det?

En interaktiv sjekkliste som hjelper lærere og skoleledere til å ta informerte valg om KI-bruk i undervisning og skoleadministrasjon.

## Hvorfor trenger vi det?

KI-verktøy som ChatGPT, oversettere og vurderingshjelpere endrer skolehverdagen. Men valg om hvilke verktøy man bruker, hvem som bruker dem, og hvor data lagres, bør ikke tas på tilfeldig grunnlag.

Denne veiviseren sikrer at du og dine kollegaer stiller de riktige spørsmålene **før** dere implementerer nye systemer.

## Hva får jeg?

✓ Spørsmål tilpasset din rolle (lærer eller skoleadministrator)  
✓ Vurderingsgrunnlag for beslutninger  
✓ Sjekklister for implementering  
✓ Ingen juridiske konklusjoner – bare fakta og prinsipper

## Viktig

- Dette verktøyet **gir vurderingsgrunnlag, ikke godkjenning**
- **Du tar beslutningen** – ikke verktøyet
- Fokus på pedagogisk verdi og ansvarlighet
- Brukt av mennesker, ikke erstatter mennesker
