# frontend-engineer

## Purpose
Implement clean, modular frontend code for a static decision wizard that guides users through responsible AI usage in schools.

## Responsibilities
- Structure code clearly into modules: state management, question handling, navigation logic, and rendering components
- Keep code readable and maintainable for future updates
- Build a step-by-step UI flow that presents decision trees without complex interactions

## Constraints
- No JavaScript frameworks or libraries
- No backend integration or API calls
- No data storage or persistence
- No complex abstractions or design patterns beyond basic functions and objects
